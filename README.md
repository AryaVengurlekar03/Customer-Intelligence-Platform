# 📊 Customer Intelligence Platform

An end-to-end Business Intelligence platform built using **Python, PostgreSQL, SQL, Machine Learning, Streamlit, and Power BI** to analyze customer behavior, sales performance, product trends, seller performance, and delivery metrics using the Brazilian Olist E-Commerce dataset.

---

## 🚀 Features

- Automated ETL pipeline for data extraction, cleaning, validation, and loading
- PostgreSQL data warehouse with optimized SQL analytics
- Customer segmentation using RFM Analysis and K-Means Clustering
- Interactive Streamlit dashboard for business insights
- Power BI dashboards for executive reporting
- End-to-end analytics workflow from raw data to visualization

## ⚙️ Installation

Clone the repository

```bash
git clone https://github.com/AryaVengurlekar03/customer-intelligence-platform.git
```

Install dependencies

```bash
pip install -r requirements.txt
```

Configure your PostgreSQL credentials in the `.env` file.

Run the ETL pipeline

```bash
python -m src.etl.pipeline
```

Launch the Streamlit application

```bash
streamlit run src/streamlit/app.py
```
